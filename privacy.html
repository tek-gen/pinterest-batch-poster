Privacy Policy

Effective date: September 16, 2025

This Privacy Policy explains how the Pinterest Batch Poster desktop application handles your data. By using this software, you agree to the practices described here.

1. No Data Collection

This application does not collect, store, or transmit any personal data from users. All operations are performed locally on your device.

2. Pinterest API Usage

The application connects to the Pinterest API using your personal access token, which you provide manually. This token is used only to:

  • Retrieve your boards
  • Post new pins to your selected boards

Your token is stored locally on your machine and is never shared externally.

3. NVIDIA or AI API Integration

If you choose to use AI-generated descriptions or tags, your selected image is sent to the corresponding AI API (e.g., NVIDIA Picasso or OpenAI) for analysis. These services are external and subject to their own privacy policies.

4. No Analytics, No Ads

This application does not use cookies, analytics tools, or advertising frameworks of any kind.

5. Local Configuration

Settings such as API tokens, selected folders, and preferences are stored on your device in a local JSON configuration file. This file is not transmitted or uploaded.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be published at this URL.
